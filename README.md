# PNGize
Convert any file to a PNG image

# Usage

## positional arguments:

* file: *file to convert*

## optional arguments:
* -h, --help:  *show this help message and exit*
* -o OUTPUT, --output OUTPUT:  *output filename*
* -x, --extract: *extract file from png*

# Requirements
* Python3
* PIL (or Pillow)
* Numpy
